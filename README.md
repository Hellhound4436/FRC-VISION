# FRC Vision

Computer vision modules developed for FIRST Robotics Competition robots.

## Features

- AprilTag Detection
- LimeLight Vision Processing
- OpenCV Integration
- Real-time Camera Feed

## Technologies

- Python
- OpenCV
- RobotPy
- NumPy

## Future Improvements

- Pose Estimation
- Multi-tag Tracking
- Performance Optimization

This repository contains vision processing code developed during my FIRST Robotics Competition (FRC) season. It is preserved as a portfolio project showcasing my experience with AprilTag detection, Limelight integration, and OpenCV.
